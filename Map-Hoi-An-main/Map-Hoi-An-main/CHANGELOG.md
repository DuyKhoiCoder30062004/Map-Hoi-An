# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-17

### Added
- Initial project setup with FastAPI backend and React frontend
- Interactive map with Leaflet integration
- AI-powered voice guidance using Google Gemini and ElevenLabs
- Multi-language support (15 languages)
- User authentication system
- Admin dashboard for restaurant management
- Real-time GPS tracking
- Audio playback with base64 encoding

### Technical Details
- Backend: FastAPI, PostgreSQL with PostGIS
- Frontend: React 19, Vite, Tailwind CSS
- AI Integration: Google Gemini API, ElevenLabs TTS
- Database: Users and Restaurants tables with geospatial data
